Conozcamos CSS

En este espacio virtual conoceremos el concepto, funcionalidad, origen y forma de uso de CCS.
Con la temática de los estilos en cascada damos la bienvenida al conocimiento de un lenguaje Web como lo es HTML.

Objetivo general

Exponer las propiedades básicas de CSS, Display: inline, block, inline-block dando claridad

Ojetivos específicos

  • Reconocer la funcionalidad de los estilos CSS y su innumerable variedad de aplicación
  • Identificar la propiedad display junto con sus tipos de aplicación tanto externa como interna
  • Aplicar las hojas de estilo en cascada de forma adecuada en diferentes realizaciones HTML
